Ano ang Filipino Camp?

Sertipikadong Programa — Ang Filipino Camp (FilCamp) ay isang makabagong interaktibong virtual kampo na espesyal na dinisenyo para sa mga mag-aaral ng Alangilan Senior High School. Pinagsama namin ang tradisyonal na pagtuturo ng wikang Filipino at makabagong teknolohiya upang lumikha ng isang natatanging karanasan sa pagkatuto.

Sa pamamagitan ng aming programa, natututo ang mga mag-aaral ng limang mahalagang makrong kasanayan — pakikinig, pagsasalita, pagbabasa, pagsulat, at panonood — sa isang masaya at nakakaengganyong paraan na nagpapalakas ng kanilang pagmamahal sa wikang Filipino.

Mga Larong Pang-makrong Kasanayan

Piliin ang kasanayan at subukan ang mga interactive na laro. Ang iyong progreso ay awtomatikong tataas.

Pagsulat

Guess the Error
  1. Basahin nang mabuti ang bawat talata.
  2. Pansinin ang bawat salita – hanapin ang maling baybay, gramatika, o pagkakasulat.
  3. Piliin ang letra (A-D) ng may error. Kung walang mali, piliin ang "D – Wala".
  4. Pindutin ang sagot bago lumipat.
  5. Bawat tamang sagot ay may puntos.
Laruin ang Guess the Error

Panonood

Viewing

Match-Up: Kilalanin ang Tauhan

  1. Manood ng palabas (Maria Clara at Ibarra).
  2. Buksan ang link, basahin ang paglalarawan.
  3. I‑match ang tamang larawan.
  4. Pindutin ang submit upang suriin.
Laruin ang Match-Up

Spin the Wheel: Heneral Luna

  1. Manood ng eksena mula Heneral Luna.
  2. Buksan ang laro, paikutin ang gulong.
  3. Piliin ang larawang tumutugma sa pangalan.
  4. Kumuha ng puntos sa tamang sagot.
Laruin ang Spin the Wheel

Pasalita

Speaking

Truth or Lie

  1. Pumili ng 2‑3 kahon.
  2. Basahin ang pangungusap sa loob.
  3. Maglahad ng opinyon kung ito ay TOTOO o HINUHA.
  4. Pangangatwiranan ang sagot.
Laruin ang Truth or Lie

Emoji Stories

  1. Pindutin ang SPIN BUTTON.
  2. Gumawa ng maikling kwento batay sa pagkakasunod-sunod ng emoji.
  3. Bumuo ng 2‑3 pangungusap.
Laruin ang Emoji Stories

Pakikinig

Listening

Makinig at Unawain

  1. Pakinggan nang mabuti ang pangungusap.
  2. Basahin ang tanong at piliin ang tamang sagot (multiple choice).
Laruin

Pakinggan Mo Ayusin Mo

  1. Makinig sa pangungusap.
  2. Ayusin ang mga ginulong salita (jumbled words).
Laruin

Pagbasa

Reading

Pag-aayos ng Letra upang Makabuo ng Salita

  1. Basahing mabuti ang ibinigay na clue o pahiwatig.
  2. Matapos basahin ang clue, tingnan ang mga nakahalong letra na ibinigay.
  3. Ayusin ang mga letra upang makabuo ng tamang salita na tumutugma sa pahiwatig.
  4. Ilagay ang bawat letra sa naaayon nitong ayos upang mabuo ang tamang salita.
  5. Ang may pinakamaraming tamang sagot ang siyang ituturing na panalo.
Laruin

Pag-unawa sa Talata

  1. Basahin nang mabuti ang bawat talata at unawain ang nilalaman nito bago sagutin ang tanong na kasunod.
  2. Pindutin ang letra ng tamang sagot mula sa dalawang pagpipilian.
  3. Piliin lamang ang titik na sa tingin ninyo ay wastong sagot.
